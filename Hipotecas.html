1. Descripción del dataset¶

Para este ejercicio se proporciona un dataset con datos sobre diferentes clientes de un banco que quieren comprar un piso, y si el banco les ha concedido la hipoteca o no.

4.2. Comprobación de la normalidad y homogeneidad de la varianza.¶

El supuesto de homogeneidad de varianzas, tambin conocido como supuesto de homocedasticidad, considera que la varianza es constante (no vara) en los diferentes niveles de un factor, es decir, entre diferentes grupos. En los modelos de regresin lineal, esta condicin de homocedasticidad suele hacer referencia a los errores (residuos) del modelo, es decir, que la varianza de los errores es constante en todas las predicciones. Cuando la condicin de homocedasticidad no se cumple, se utiliza el trmino heterocedasticidad.

Existen diferentes test que permiten evaluar si las observaciones proceden de poblaciones con la misma varianza. Todos ellos consideran como hiptesis nula que la varianza es igual entre los grupos y como hiptesis alternativa que no lo es. La diferencia entre ellos es el estadstico de centralidad que utilizan:

*Los test que utilizan la media de la varianza son los que mayor poder estadstico consiguen cuando los datos siguen distribuciones normales.

*Los test que utilizan la mediana de la varianza consiguen mejores resultados cuando los datos siguen distribuciones asimtricas o se alejan de la normalidad.

Por lo general, si no se puede alcanzar cierta seguridad de que las poblaciones que se comparan son de tipo normal, es recomendable recurrir a test que comparen la mediana de la varianza.

En nuestro caso, los dos grupos que srn sometidos a anlisis sern el conjunto de clientes a los que se les concede hipotecas y el conjunto de clientes a los que no se les concede hipoteca. Concretamente, se estudiar para la variable "ingresos", la cul, cmo se puede observar en el histograma extrado anteriormente para esta variable, no sigue una distribucion normal. Con lo cul se usar un test que utilize la mediana de la varianza, concretamente, el test de Levene.

Fuente: https://www.cienciadedatos.net/documentos/pystats07-test-homocedasticidad-heterocedasticidad-python.html

El objetivo del estudio es determinar si, la diferencia observada de 12.537,80 unidades, est dentro de lo que cabra esperar debido nicamente al muestreo aleatorio, si no existiera diferencia real entre las poblaciones ( H0 : todas las observaciones proceden de la misma poblacin).

Si se obtuviera una nueva muestra de clientes, la diferencia promedio entre grupos sera ligeramente distinta, aunque no existiera una diferencia real entre hipotecados y no hipotecados. Por lo tanto, dar respuesta a este problema pasa por determinar cunta diferencia se espera por el simple hecho de repetir el muestreo.

Mediante bootstrapping, se generan nuevas pseudo-muestras de individuos empleando las dos muestras originales (hipotecados y no hipotecadis) combinadas. Con cada una de las pseudo-muestras, se generan dos grupos de tamaos iguales a los grupos de las muestras originales y se calcula la diferencia del estadstico, en este caso la media. Para ello, en cada iteracin y antes del reparto en grupos, se genera una nueva pseudo-muestra que tiene el mismo tamao que la muestra original pero formada por distintas observaciones.

def calcular_estadistico(x):
    '''
    Funcin para calcular el estadstico de inters.
    
    Parameters
    ----------
    x : numpy array
         valores de la muestra.
         
    Returns
    -------
    estadstico: float
        valor del estadstico.
    '''
    estadistico = np.mean(x)
    
    return(estadistico)


def bootstraping_2_grupos(x1, x2, fun_estadistico, n_iteraciones=9999):
    '''
    Funcin para calcular la diferencia de un estadstico entre dos grupos en
    mltiples muestras generadas mediante muestreo repetido con reposicin
    (bootstrapping).
    
    Parameters
    ----------
    x1 : numpy array
         valores de la muestra del grupo 1.
         
    x2 : numpy array
         valores de la muestra del grupo 2.
 
    fun_estadistico : function
        funcin que recibe como argumento una muestra y devuelve el valor
        del estadstico.
        
    n_iteraciones : int
        nmero iteraciones (default `9999`).
        
    Returns
    -------
    distribuciones: numpy array
        diferencia entre ambos grupos en cada muestra de bootstrapping.
    '''
    
    n1 = len(x1)
    n2 = len(x2)
    pool = np.hstack((x1, x2))
    dist_boot = np.full(shape=n_iteraciones, fill_value=np.nan)
    
    for i in tqdm(range(n_iteraciones)):
        # Se crea una nueva muestra
        resample = np.random.choice(pool, size=n1+n2, replace=True)
        # Se reparten las observaciones en dos grupos y se calcula el estadstico
        estadistico_1 = fun_estadistico(resample[:n1])
        estadistico_2 = fun_estadistico(resample[n1:])
        # Diferencia entre estadsticos
        dist_boot[i]  = estadistico_1 - estadistico_2
        
    return dist_boot
